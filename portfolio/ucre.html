About the Project

For our User Centered Research & Evaluation (UCRE) class, we covered a number of HCI techniques which we applied to various projects within as well as outside the class.

Techniques Covered

  • Think Aloud
  • Interview
  • Observation
  • Heuristic Evaluation
  • Cognitive Walkthrough
  • Affinity Diagramming
  • Competitive Analysis
  • Log Analysis
  • Model Consolidation
  • Surveys
  • Contextual Inquiry
  • Visioning
