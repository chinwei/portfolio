View Project

Role: Frontend Development

As this was a government project targetted at the general public, it had to be widely accessible across a range of browsers. I worked with a web designer to design and build a solution that was appealing to users on modern systems, but remained functional on older browsers (up to Internet Explorer 7). I built this website on Serve, a Ruby-based backend, using HAML/SASS/Javscript and deployed as a static HTML package.
